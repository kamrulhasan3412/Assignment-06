### 6. Answer of all questions:

1. What is the difference between **var**, **let**, and **const**?

   **_Answer :_**

   **The difference between var, let, and const is :**

   **var** is function-scoped and can be redeclared and updated, **let** is block-scoped and can be updated but not redeclared, and **const** is block-scoped and cannot be updated or redeclared

2. What is the difference between **map()**, **forEach()**, and **filter()**?

   **_Answer :_**

   **The difference between map(), forEach(), and filter() is :**

   **map()** returns a new array by transforming each element, **forEach()** simply loops through the elements and does not return a new array, and **filter()** returns a new array containing only the elements that meet a specified condition.

3. What are **arrow functions** in ES6?

   **_Answer :_**

   **Arrow functions** in ES6 are a **shorter** and **simpler** way to write functions, and they automatically use the value from the place where they were written.

4. How does **destructuring assignment work** in ES6?

   **_Answer :_**

   **Destructuring** in **ES6** is a quick way to take values from arrays or objects and put them into variables.

5. Explain **template literals** in ES6. How are they different from **string concatenation**?

   **_Answer :_**

   **Template literals** in ES6 let you easily create strings and add variables **using ${}**, instead of joining strings with +.
